# IYF Season 10 - Week 1 Tasks (Basic HTML)

## Overview
Basic HTML tasks demonstrating fundamental web development skills without styling.

## Tasks

### 1. Personal Portfolio
- **File**: `index.html`
- **Description**: Basic personal portfolio page with introduction, hobbies, and contact information
- **Skills**: Basic HTML structure, semantic tags, links

### 2. Multi-Page Portfolio
- **Files**: 
  - `portfolio-index.html` - Portfolio homepage
  - `about.html` - About page with education and skills
  - `portfolio.html` - Projects showcase
  - `contact.html` - Contact information and form
- **Description**: Complete multi-page portfolio website
- **Skills**: Multi-page architecture, navigation, semantic HTML5

### 3. Contact Form
- **File**: `contact-form.html`
- **Description**: Simple contact form with validation
- **Skills**: Forms, input types, validation attributes

### 4. Semantic HTML Conversion
- **File**: `semantic-html.html`
- **Description**: Conversion to semantic HTML5 elements
- **Skills**: Semantic markup, proper HTML5 structure

### 5. Box Model Practice
- **File**: `box-model-practice.html`
- **Description**: CSS box model demonstration
- **Skills**: Basic CSS, margins, padding, borders

### 6. DevTools Exploration
- **File**: `devtools-exploration.md`
- **Description**: Analysis of three different websites using browser developer tools
- **Skills**: Web inspection, performance analysis, code analysis

## How to View
1. Open any HTML file in your web browser
2. Navigate between multi-page portfolio files
3. No additional setup required - pure HTML only

- Semantic HTML5 usage
- Introduction to CSS box model
- Browser developer tools usage
